
# Как объяснить железяке, что ты от неё хочешь
Давайте разберём код, который заставит железку сказать "Hello, World!"  

```cpp
#include <iostream>  // Магическое заклинание для подключения суперспособности "вывод текста"

using namespace std; // Разрешаю называть моего кота просто "Барсик", а не "Уважаемый Барсик из рода std"

int main() {         // Стартовая точка, как "Старт!" в Mario Kart
    cout << "Hello, World!" << endl;  // Выводим текст и бьём Enter ударом каратэ
    return 0;        // Говорим ОС: "Всё ок, не паникуй!" (0 = код успеха, любой другой = «ой, всё»)
}
```
Разбор нашей писанины!:

- `#include <iostream>` — с помощью директивы `#include` мы подключаем стандартную библиотеку `iostream` для ввода и вывода. Писать код без библиотек — это как перед каждой поездкой на автобусе собирать его с нуля, причём начиная с добычи железной руды для деталей. В дальнейшем мы ещё познакомимся с другими библиотеками и способами ввода и вывода
- `using namespace std` — разрешаем себе лениться. Без этой строки пришлось бы писать `std::cout`, будто здороваться с другом по имени-отчеству. Эта строка необязательная, но с ней проще и приятнее
- `int main() {` — главная функция. Она похожа на сцену в театре: всё действие происходит между `{` и `}`. Даже если ваша программа — рок-опера на 10 тысяч строк, выполнение начинается тут. Подробнее про функции мы будем говорить позднее, пока важно запомнить: всё, что написано после открывающей фигурной скобки и до закрывающей, это содержимое функции
- `cout << "Hello, World!" << endl;` — оператор `<<` — буквально стрелка «туда». Представьте, что вы кидаете текст в чёрную дыру консоли, которую зовут `cout`. Туда же отправляется `endl` — `Enter` на стероидах, он выполняет перенос строки и сброс буфера
- `return 0;` — код возврата. 0 = «всё супер», 1+ = «ой, кое-что пошло не так... но это секрет»
- закрывает нашу программу фигурная скобка, она показывает, что мы закончили этот блок кода


```
Интересный факт: первый "Hello, World!" написали Керниган и Ритчи в своём учебнике по языку `C` (без плюсов) ещё в 1978 году.
```

Предлагаем не останавливаться на первой программе и сразу же написать вторую:

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Лучше " << "гор " << "могут " << "быть " << "только " << "горы " << endl;
    cout << "\tВысота Эльбруса: " << 5642 << " м\n";  // \t — это как Tab в калькуляторе
    cout << "\tВысота Эвереста: " << 8849 << " м\n";  // \n — Enter для бедных
    cout << "На которых ещё не бывал"<< endl;
    return 0;
}
```

Вот что увидит пользователь, когда запустит эту программу:

Лучше гор могут быть только горы 
	Высота горы Эльбрус 5642 метра
	Высота горы Эверест 8849 метров
На которых ещё не бывал

Фишки кода:

- с помощью `cout` можно за раз выводить на экран несколько строк(в нашем случае — слов). Если мы хотим выводить их по правилам русского языка, то есть с пробелами между словами, нужно эти пробелы явно указать. Строка из нашей программы выведет на экран `Лучше гор могут быть только горы`, но если убрать пробелы, появится что-то непонятное: `Лучшегормогутбытьтолькогоры` (выглядит как название вулкана в Исландии)
- в строках могут быть специальные символы:

- `\t` — табуляция, отступ для ленивых
- `\n` — перенос строки. `endl` круче, но `\n` работает быстрее (как фастфуд vs ресторан)

- можно выводить не только строки, но и числа без кавычек — компьютер поймёт, что это не текст
