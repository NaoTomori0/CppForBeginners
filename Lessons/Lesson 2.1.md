# Структура программы на C++
1. Директивы препроцессора (`#include` В самом верху подключаются библиотеки. Без них программа не будет знать, как выводить текст или работать с математикой.
2. Объявления и пространства имен: Здесь подключаются пространства имен (например, `using namespace std;`, хотя в больших проектах лучше писать `std::` явно) или объявляются глобальные переменные и прототипы функций.
3. Главная функция (`int main()`): Это «сердце» программы. Любой код начинает выполняться именно отсюда. Если этой функции нет, программа не запустится.
4. Тело функции (внутри `{ }`): Здесь пишутся инструкции, команды, расчеты и вызовы других функций. Каждая инструкция заканчивается точкой с запятой `;`.
5. Возврат значения (`return 0;`): Завершающий аккорд, который говорит операционной системе: «Все прошло по плану, ошибок нет».

**Наглядный пример:**

``` cpp
// 1. Подключение библиотек
#include <iostream> 

// 2. Глобальные настройки (необязательно)
using namespace std; 

// 3. Точка входа в программу
int main() { 
    
    // 4. Тело программы (логика)
    cout << "C++ Program" << endl; 

    // 5. Завершение
    return 0; 
}
```


Наша программа это последовательность инструкций для процессора, и их еще можно группировать фигурными скобками (`{ }`), вот по сути что такое наша программа.

Пример группировки:
``` cpp
int main() { 
    ...
    {
	    ...
    }
    {...}
    return 0; 
}
```
